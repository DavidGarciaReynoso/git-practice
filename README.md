# Learning Git

A practice repository created to learn and reinforce **Git** and **GitHub** concepts.

## Overview

This repository was created **only for practice** while following the **Amigoscode Git and Github Essentials** course.

It is not intended to be a complete software project. This repository was created **only to practice Git and GitHub**.  
Any project files inside it are included only to generate changes and simulate real workflow practice.

## Repository Purpose

The main purpose of this repository is to practice:

- creating and managing a Git repository
- tracking file changes
- staging updates
- making commits
- editing commit history
- connecting a local repository to GitHub
- authenticating with SSH
- pushing and pulling changes

## Topics Practiced

- Git fundamentals
- Git installation and setup
- verifying the installation
- Git configuration
- repository initialization
- `git add`
- commits
- amending commit messages
- GitHub basics
- creating a GitHub repository
- SSH key setup
- `git push`
- `git pull`
- branches
- pull requests
- merging
- conflict resolution
- rebasing
- general Git/GitHub workflow improvements
